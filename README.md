# Miami DDA — Landing Page

A bespoke, single-page position paper prepared for the Miami Downtown Development Authority at eMerge Americas 2026 (Booth 417). The page reads as a civic memorandum: institutional layout, Miami DDA's teal accent, stepped-skyline motif, Helvetica Neue type, no startup-deck aesthetic. It carries the pitch body from the fit analysis and the CTA to Matt Creamer's 30-minute Calendly.

Intended URL path: /for/miami-dda

## Files

- `index.html` — Structure, copy, and semantic markup.
- `styles.css` — Institutional brand tokens from `brand-pack.json` (teal #1BB5C9, accent orange used sparingly, Helvetica Neue stack).
- `logo.png` — Miami DDA primary logo.
- `logo-horizontal.webp` — Wordmark lockup (not currently rendered; kept for future swap).

## Notes

- Eduba branding appears only in the footer and in named case-study references, per the shared chrome rules.
